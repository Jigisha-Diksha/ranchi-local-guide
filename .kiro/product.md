# Ranchi Local Guide

## Product Overview
A translator application for local Ranchi slang, food terms, and cultural phrases. Built with Kiro AI for the AI For Bharat Hackathon.

## Key Features
- Translates Ranchi local slang to standard Hindi/English
- Food terminology specific to Ranchi cuisine
- Cultural phrases and expressions
- Real-time translation using Kiro AI

## Target Audience
- Tourists visiting Ranchi
- New residents in Ranchi
- Anyone interested in Ranchi culture
- Food enthusiasts exploring local cuisine

## Technology Stack
- Kiro AI for language processing
- Python Flask for web application
- HTML/CSS for user interface

## Use Cases
1. Understanding local food menus
2. Communicating with locals
3. Learning Ranchi cultural expressions
4. Exploring traditional Ranchi terminology

## Sample Translations
- Local food terms (Litti Chokha, Dhuska, Pittha)
- Common phrases and greetings
- Cultural expressions
- Slang and colloquialisms

## Future Enhancements
- Voice input/output
- Mobile application
- Expanded vocabulary database
- Regional dialect variations
